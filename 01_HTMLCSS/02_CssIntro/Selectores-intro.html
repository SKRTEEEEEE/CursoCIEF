Selectores CSS

Selectores especiales

.primerDiv p

Esto afectaria a todos los elementos "p" dentro de la clase .primerDiv

.priverDiv > p

Esto solo afectaria a los elementos "p" que son hijos directos de la clase .primerDiv

Selectores de seudoclase

Afectan al realizarse ciertas acciones, hover, etc...

Prioridades en los selectores CSS

  1. Selector de id( # )
  2. Selector de clase( . )
  3. Selector de etiqueta( div, p, ... )

Peso para las prioridades CSS

El peso se usa para que selector tendra prioridad sobre otros selectores, dentro de la misma hoja de estilos

  • Selector id -> 100
  • Selector clase -> 10
  • Selector etiqueta -> 1

Cuando hacemos selectores "especiales", con mas de un selector, estos se suman

En VSC, en la hoja de estilos (CSS), si dejamos el mouse encima del selector, podemos ver los pesos
